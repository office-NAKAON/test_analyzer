# 得点分析＆二極化ビジュアライザー

テストの得点データを入力するだけで、分布の可視化・統計分析・学習指導の提案を行う Web アプリです。生徒名や番号は不要で、点数のみで分析できます。

## 主な機能

- **テスト情報の入力** — 教科名、テスト名、満点を設定
- **点数の一括入力** — 最大 110 人分。Excel などからの貼り付けにも対応
- **統計データの表示** — 平均点、中央値、標準偏差、最高点、最低点
- **グループ別分析** — 上位・下位グループの人数をスライダーで調整し、平均点・得点率を表示
- **グラフ表示**
  - 度数分布（上位・下位・中間層を色分け）
  - 順位順グラフ
- **二極化の判定** — 得点分布から二極化の傾向を自動判定
- **学習指導の提案** — 数値から読み取れる傾向と、今後のアプローチ案を自動生成
- **印刷・PDF 保存** — 分析結果画面をそのまま印刷可能
- **入力内容の自動保存** — ブラウザの localStorage に保存（再読み込み後も復元）

## プライバシー

入力した点数は**外部サーバーへ送信されません**。すべてブラウザ内で処理されます。

## 使い方

1. `index.html` をブラウザで開く
2. 教科名・テスト名・満点を入力する
3. 各生徒の点数を入力する（または「点数をまとめて貼り付ける」から一括入力）
4. 「分析する」をクリックする
5. 分析画面でスライダーやグラフの切り替えを使い、結果を確認する

「サンプルを入力」ボタンで、デモ用のデータをすぐに試せます。

## 動作環境

- モダンブラウザ（Chrome、Edge、Firefox、Safari など）
- サーバーやビルドツールは不要（静的 HTML のみ）
